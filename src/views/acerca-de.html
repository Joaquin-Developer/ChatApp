Acerca de SafeText

SafeText es una aplicación web de mensajería sencilla, enfocada en la privacidad de sus usuarios.

En su versión BETA (V.1.0) cuenta únicamente con una aplicación web de pequeño porte. Próximamente se desarrollarán mejoras en el sistema, y se lanzará una aplicación móbil para Android.

Datos técnicos de la aplicación:

Desarrollo en el frontend:

En la versión 1.2 se implementará Angular como framework.
Desarrollo en el backend: Node.js
Próximamente se implementará persistencia de datos de usuarios con MongoDB

Nota: ¡Los mensajes no se almacenan en el servidor de la aplicación! Mantenemos tu privacidad a salvo.
